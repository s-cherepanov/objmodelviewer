------FAQ:
Q: After importing the object, program simply disappears/crash.
A: Check if all files related to the obj file are in same place, especially the mtl one.

Q: After importing the object, I dont see materials/textures.
A: Check addresses in your mtl file and UVW mapping.

Q: Is this software open source?
A: Yes it is, you can download the source code at 
   http://sourceforge.net/projects/objmodelviewer/
